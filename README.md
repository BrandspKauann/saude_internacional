# Saúde Internacional

Site institucional e blog da Saúde Internacional, com foco em planos e seguros de saúde internacionais para estrangeiros, expatriados, estudantes e famílias.

## Comandos

```bash
npm start
```

Executa o servidor local em `http://localhost:4174`.

```bash
npm run build
```

Copia o conteúdo de `site/` para `dist/`.
